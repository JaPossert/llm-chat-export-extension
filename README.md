# LLM Conversation Exporter

Export conversations from ChatGPT, Claude, Gemini, Grok, and **Lumo** to plain text files. All processing happens locally in your browser — nothing leaves your machine.

## Supported Platforms
- ChatGPT (OpenAI)
- Claude (Anthropic)
- Gemini (Google)
- Grok (X)
- **Lumo (Proton)** — Proton's privacy-first AI with a cat mascot 🐱

---

## 🚀 Install (no technical knowledge needed)

### Step 1 — Download the extension

1. On this page, click the green **Code** button (top right area)
2. Click **Download ZIP**
3. A file called `llm-chat-export-extension-main.zip` will download to your Downloads folder

### Step 2 — Unzip it

- **Mac:** Double-click the ZIP file — a folder appears next to it
- **Windows:** Right-click the ZIP file → "Extract All" → click Extract

### Step 3 — Load it into Chrome

1. Open **Chrome** (or Arc, Brave, or Edge — any Chromium browser works)
2. In the address bar, type `chrome://extensions` and press Enter
3. In the top-right corner, turn on **Developer mode** (the toggle switch)
4. Click the **Load unpacked** button that appears
5. Select the folder you unzipped in Step 2
6. The extension icon will appear in your browser toolbar (puzzle piece icon → pin it)

### Step 4 — Export a conversation

1. Go to any supported chat (e.g. [lumo.proton.me](https://lumo.proton.me), [claude.ai](https://claude.ai), [chatgpt.com](https://chatgpt.com))
2. Open a conversation
3. Click the extension icon in your toolbar
4. Click **Export Conversation**
5. A `.txt` file downloads to your computer

## 📖 Usage (quick reference)

1. Navigate to a supported AI chat and open a conversation
2. Click the extension icon
3. Click "Export Conversation"
4. Done — the conversation saves as a text file

## Privacy & Security

- **No data collection**
- **Local processing only**
- **No analytics**
- **Open source**

Nothing is sent anywhere. The extension reads the page you're already on and saves the text to your own computer.
